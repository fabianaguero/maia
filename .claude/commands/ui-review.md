Actuá como un experto en UX Design y accesibilidad. Revisá el componente
o pantalla que te indico con estos criterios específicos para Maia:

**1. Cognitive Load**
- ¿Cuántos conceptos nuevos necesita entender el usuario antes de poder actuar?
- ¿Hay terminología que requiere conocimiento previo de DJ/producción musical?
- Contá la cantidad de elementos interactivos visibles en el primer scroll.

**2. Jerarquía visual**
- ¿Está claro cuál es la acción principal?
- ¿El usuario sabe qué hacer en menos de 5 segundos?
- ¿Hay competencia visual entre elementos de distinta importancia?

**3. Terminología**
- Marcá cada término que un usuario de Spotify (no DJ) no entendería.
- Sugerí reemplazos en español neutro o inglés técnico-SRE.

**4. Feedback del sistema**
- ¿El usuario sabe si el sistema está funcionando?
- ¿Los estados de carga/error son claros?
- ¿Hay confirmación visual de las acciones exitosas?

**5. Accesibilidad básica**
- ¿Hay contraste suficiente (WCAG AA mínimo)?
- ¿Los elementos interactivos tienen labels accesibles?
- ¿Funciona con teclado?

Entregá el resultado como:
- Lista priorizada de problemas (P1/P2/P3)
- Sugerencia de fix concreta para cada P1
- Estimación de esfuerzo (XS/S/M/L)
